Pairhub · 系统架构图 v0.1

微信小程序端 + Node.js 后端 + PostgreSQL + Redis + 微信开放接口 — 2026-06-02

前端 (WeChat MiniProgram)

  • • WXML + WXSS + TypeScript
  • • 微信原生 API (login/getLocation)
  • • 腾讯地图 SDK 选点
  • • Web H5 fallback (P2)

后端 (Node 20 + Fastify 4)

  • • 模块化 service 拆分
  • • Prisma ORM + 迁移
  • • JWT 鉴权 (HS256)
  • • pnpm workspace (后续可拆 monorepo)

数据层

  • • PostgreSQL 16 + Decimal 坐标
  • • Redis 7 缓存 + 限流
  • • Prisma 事务保证报名一致性
  • • 地理筛选用 Haversine (M2 PostGIS)

外部依赖

  • • 微信开放平台 (登录/内容安全/订阅消息)
  • • 腾讯地图 API (GCJ-02)
  • • GitHub Actions (CI/CD)
  • • 部署目标: 腾讯云 (M2 阿里云灾备)
