## 0. 设计原则

1. **RESTful 资源命名**：复数名词，路径不出现动词
2. **JSON 优先**：所有请求/响应 JSON，UTF-8
3. **无状态**：每个请求带完整鉴权信息（JWT）
4. **幂等性**：写操作支持 `Idempotency-Key` 头（M2 引入）
5. **可缓存**：GET 响应含 `ETag`（M2 引入）
6. **跨端一致**：后端 100% 共用，三端 0 业务规则漂移
7. **类型安全**：所有 endpoint 配 TS interface，OpenAPI 自动生成（M1-W2 落地）

## 2. 通用请求 / 响应 Schema

### 2.1 请求头

| 头 | 必填 | 说明 |
|----|------|------|
| `Content-Type: application/json` | 写操作 | UTF-8 |
| `Authorization: Bearer <jwt>` | 鉴权接口 | 来自 `/api/v1/auth/*-login` |
| `X-Request-Id: <uuid>` | 推荐 | 服务端日志追踪，原样回显 |
| `Accept-Language: zh-CN \| en-US` | 可选 | 国际化（M2） |
| `Idempotency-Key: <uuid>` | 可选 | 写操作幂等性（M2） |

### 2.2 统一响应封装

```typescript
// 成功响应 - 单个资源
interface ApiResponse<T> {
  data: T;
  meta?: {
    requestId: string;
    timestamp: string;  // ISO 8601
  };
}

// 成功响应 - 列表（带分页）
interface ApiListResponse<T> {
  data: T[];
  pagination: {
    page: number;        // 当前页，从 1 开始
    page_size: number;   // 每页大小
    total: number;       // 总数
    has_more: boolean;   // 是否还有下一页
  };
  meta?: {
    requestId: string;
    timestamp: string;
  };
}
```

```json
  }
}
```

```json

    { "id": "act_001", "title": "..." },
```

### 2.3 错误响应（RFC 7807 Problem Details）

```typescript
interface ApiError {
  type: string;         // 错误类型 URL（指向文档）
  title: string;        // 简短标题
  status: number;       // HTTP 状态码
  detail?: string;      // 详细描述（可对用户展示）

  errors?: Array<{      // zod 校验错误时填
```

## 3. 鉴权（JWT）使用规则

### 3.1 JWT 格式

```typescript
// 签发：HS256, 7 天过期
interface JwtPayload {
  sub: string;          // userId（内部 cuid）
  providers: AuthProvider[];  // ['WECHAT', 'APPLE', 'GOOGLE']
  primary: AuthProvider;      // 'WECHAT' | 'APPLE' | 'GOOGLE'
  region: 'CN' | 'GLOBAL';   // 用户所在地域（v1.1 起）
  iat: number;          // issued at
  exp: number;          // expiry
  iss: 'studybuddy-api';
  aud: 'studybuddy-client';
}

type AuthProvider = 'WECHAT' | 'APPLE' | 'GOOGLE';
```

### 3.2 鉴权规则

| 接口 | 鉴权要求 |
|------|----------|
| `GET /api/v1/health` | 无 |
| `GET /api/v1/activities` | 无（公开） |
| `GET /api/v1/activities/:id` | 无（公开） |
| `GET /api/v1/activities/:id/participants` | 无（公开） |
| `POST /api/v1/auth/*-login` | 无 |
| `POST /api/v1/auth/refresh` | Refresh token（v0.3） |
| `POST /api/v1/activities` | ✅ JWT |
| `PATCH /api/v1/activities/:id` | ✅ JWT + **必须是 creator** |
| `DELETE /api/v1/activities/:id` | ✅ JWT + **必须是 creator** |
| `POST /api/v1/activities/:id/signup` | ✅ JWT |
| `DELETE /api/v1/activities/:id/signup` | ✅ JWT（且必须已报名） |
| `GET /api/v1/users/me/*` | ✅ JWT |
| `PATCH /api/v1/users/me` | ✅ JWT |
| `DELETE /api/v1/users/me` | ✅ JWT |
| `POST /api/v1/auth/link-provider` | ✅ JWT |

### 3.3 错误处理

| 情况 | HTTP | code |
|------|------|------|
| 缺 `Authorization` 头 | 401 | `UNAUTHORIZED` |
| `Bearer` 后无 token | 401 | `UNAUTHORIZED` |
| token 签名错误 | 401 | `INVALID_TOKEN` |
| token 过期 | 401 | `TOKEN_EXPIRED` |
| token 有效但 sub 用户不存在/被封禁 | 401 | `USER_NOT_FOUND` / `USER_BANNED` |
| 已登录但权限不足（如非 creator 改活动） | 403 | `FORBIDDEN` |

### 3.4 安全细则

- ❌ **JWT 不存于 Cookie**：避免 CSRF，统一用 `Authorization: Bearer` 头
- ✅ **HTTPS 强制**：生产 HTTP 全部 301 → HTTPS
- ✅ **Token 泄露应对**：v0.3 引入 refresh token + 黑名单（M2）
- ✅ **客户端存储**：
  - 小程序：`wx.setStorageSync('jwt', token)`
  - Flutter：`flutter_secure_storage`

## 4. 错误码体系（全集）

### 4.1 4xx 客户端错误

| HTTP | code 常量 | 含义 | 触发场景 |
|------|-----------|------|----------|
| 400 | `VALIDATION_ERROR` | 请求参数校验失败 | zod 校验失败，附 `errors[]` |
| 400 | `INVALID_JSON` | 请求体不是合法 JSON | JSON 解析失败 |
| 400 | `MISSING_HEADER` | 缺少必填头 | 必填 header 缺失 |
| 401 | `UNAUTHORIZED` | 未提供 token | 缺 `Authorization` 头 |
| 401 | `INVALID_TOKEN` | token 签名/格式错误 | JWT 解析失败 |
| 401 | `TOKEN_EXPIRED` | token 过期 | JWT `exp` 过期 |
| 401 | `INVALID_CODE` | 微信 code 无效 | jscode2session 返回 40029 |
| 401 | `INVALID_ID_TOKEN` | Apple/Google id_token 无效 | verifyIdToken 失败 |
| 401 | `WX_UNAVAILABLE` | 微信服务暂不可用 | jscode2session 返回 -1（重试 1 次后） |
| 403 | `FORBIDDEN` | 权限不足 | 非 creator 改活动 / 非参与者退出 |
| 403 | `USER_BANNED` | 用户被封禁 | 状态 BANNED |
| 403 | `CONTENT_BLOCKED` | 内容被审核拦截 | contentCheck = BLOCKED |
| 404 | `USER_NOT_FOUND` | 用户不存在 | sub 找不到 |
| 404 | `ACTIVITY_NOT_FOUND` | 活动不存在 | id 找不到 |
| 404 | `SIGNUP_NOT_FOUND` | 报名记录不存在 | 取消未报名活动 |
| 409 | `CONFLICT` | 资源冲突（通用） | 乐观锁冲突 |
| 409 | `ALREADY_SIGNED_UP` | 重复报名 | 同一 (activity, user) 已存在 |
| 409 | `PROVIDER_ALREADY_LINKED` | openid/sub 已被占用 | 合并冲突 |
| 422 | `BUSINESS_RULE_VIOLATION` | 业务规则违反 | 通用 |
| 422 | `ACTIVITY_NOT_OPEN` | 活动未开放报名 | status != RECRUITING/FULL |
| 422 | `ACTIVITY_NOT_CANCELABLE` | 活动不可取消 | status = STARTED/ENDED |
| 422 | `ACTIVITY_FULL` | 活动已满员 | current_count = max |
| 422 | `INVALID_TRANSITION` | 状态机迁移非法 | 状态机校验失败 |
| 422 | `CANCEL_TOO_LATE` | 取消活动过晚 | now > start_time - 1h |
| 422 | `LAST_PROVIDER` | 至少保留一种登录方式 | 解绑到 0 |
| 429 | `RATE_LIMIT_EXCEEDED` | 通用限流 | IP / 用户限流 |
| 429 | `LOGIN_RATE_LIMITED` | 登录限流 | 登录 10/min/IP |
| 429 | `CREATE_RATE_LIMITED` | 创建活动限流 | 用户 10/h |
| 429 | `SIGNUP_RATE_LIMITED` | 报名限流 | 用户 30/h |
| 429 | `WX_RATE_LIMITED` | 微信 API 限流 | jscode2session 45011 |

### 4.2 5xx 服务端错误

| HTTP | code 常量 | 含义 | 触发场景 |
|------|-----------|------|----------|
| 500 | `INTERNAL_ERROR` | 通用服务器错误 | 未捕获异常 |
| 500 | `DATABASE_ERROR` | 数据库错误 | Prisma 抛出 |
| 500 | `REDIS_ERROR` | 缓存错误 | Redis 不可用 |
| 502 | `WX_UNAVAILABLE` | 微信服务不可用 | jscode2session -1 重试后仍失败 |
| 502 | `MAP_API_ERROR` | 地图服务错误 | 腾讯地图 API 失败 |
| 502 | `PUSH_FAILED` | 推送失败 | APNs / FCM 错误（不阻塞业务） |
| 503 | `SERVICE_UNAVAILABLE` | 服务暂不可用 | 维护中 / 过载 |
| 504 | `TIMEOUT` | 请求超时 | 上游 / DB 慢查询 |

### 4.3 错误码使用规则

- ✅ 客户端必须**根据 `code` 做分支**，不根据 `status`（除 200/非 200 区分）
- ✅ 错误文案**展示 `detail`**（已国际化 / 用户友好）
- ✅ 业务码命名格式：`<RESOURCE>_<REASON>`（如 `ACTIVITY_FULL`）
- ❌ 错误码**永不重用**：废弃时标记 `[DEPRECATED]`，不删除

## 5. 分页、排序、筛选

### 5.1 分页

```typescript
// 请求参数
interface PaginationQuery {
  page?: number;       // 默认 1，最小 1
  page_size?: number;  // 默认 20，最大 100
}

// 响应（已定义于 §2.2）
interface Pagination {
  page: number;
  page_size: number;
  total: number;
  has_more: boolean;
}
```

- 默认 `page=1, page_size=20`
- `page_size > 100` → 自动 clamp 到 100，**不报错**
- `page` 超出范围 → 返回空 `data: []`，`has_more: false`

### 5.2 排序

```typescript
type ActivitySort = 'start_time_asc' | 'start_time_desc' | 'created_desc' | 'distance_asc';
```

| 值 | 含义 | 备注 |
|----|------|------|
| `start_time_asc` | 活动开始时间升序（默认） | 最近的在前 |
| `start_time_desc` | 活动开始时间降序 | 远的在前 |
| `created_desc` | 创建时间降序 | 最新发布在前 |
| `distance_asc` | 距离升序 | 需传 `lat` + `lng` |

### 5.3 筛选（仅 `/api/v1/activities`）

```typescript
interface ActivityListQuery extends PaginationQuery {
  // 类型
  type?: 'STUDY' | 'SPORTS' | 'BOARD_GAME' | 'ONLINE_GAME' | 'OTHER';

  // 状态（多选）
  status?: ActivityStatus[];   // ['RECRUITING', 'FULL', 'STARTED']

  // 时间
  start_after?: string;        // ISO 8601, start_time >= this
  start_before?: string;       // ISO 8601, start_time <= this

  // 地理（需同时传）
  lat?: number;                // -90 ~ 90
  lng?: number;                // -180 ~ 180
  radius_km?: number;          // 0.1 ~ 50

  // 关键字
  keyword?: string;            // 标题/描述模糊搜索，≤ 50 字

  // 标签
  tags?: string[];             // OR 匹配

  // 排序
  sort?: ActivitySort;
}
```

**默认筛选**：
- `status`: `['RECRUITING', 'FULL', 'STARTED']`（隐藏 ENDED / CANCELED）
- `content_check`: 仅 `PASS`（隐藏 BLOCKED）
- `sort`: `start_time_asc`

**地理筛选**（Haversine 公式）：
- 必须同时传 `lat` + `lng` + `radius_km`
- 距离结果附在响应 `location.distance_km` 字段

## 7. 限流

### 7.1 规则表

| 维度 | 阈值 | 错误码 |
|------|------|--------|
| 全局 IP | 100 req/min | 429 RATE_LIMIT_EXCEEDED |
| 登录接口 IP | 10 req/min | 429 LOGIN_RATE_LIMITED |
| 创建活动用户 | 10 次/小时 | 429 CREATE_RATE_LIMITED |
| 报名活动用户 | 30 次/小时 | 429 SIGNUP_RATE_LIMITED |

### 7.2 实现

- **Redis 滑动窗口**（ZSET 存请求时间戳）
- 响应头：`X-RateLimit-Limit`、`X-RateLimit-Remaining`、`X-RateLimit-Reset`
- 超限响应头：`Retry-After: <秒数>`

### 7.3 限流例外

- 内部健康检查 `/api/v1/health`：**不限流**（否则监控会假阳性）
- 管理员接口（M3）：独立白名单

## 9. 版本与兼容

### 9.1 版本策略

- URL 前缀：`/api/v1/`（v0.2 暂用 `/api/`，v1 起统一加版本）
- **兼容性策略**：非破坏性增强向后兼容；破坏性变更走 `/api/v2/`
- 客户端强推 `Accept: application/json; version=1`

### 9.2 破坏性变更定义

任何以下变更视为破坏性，必须发 v2：

- 删除字段
- 字段类型变更（如 `number` → `string`）
- 必填字段新增
- 错误码语义变更
- 鉴权方式变更

非破坏性：

- 新增可选字段
- 新增 endpoint
- 新增错误码（客户端按 code 分支不会 break）
- 文档/注释更新
