## 1. Pairhub 是什么？

Pairhub 是一款面向**海外中国留学生**的「同好/活动匹配」App。
你按地理位置、兴趣标签、学校、专业搜索附近正在发生的活动——
学习小组、运动局、桌游夜、约饭、短期旅行——一键报名、签到、活动后
互评。

**当前可用平台**：

- iOS / Android（Flutter，App Store / Google Play / 国内应用市场）
- 微信小程序（搜索「Pairhub」）

**主要功能**：

- 地图找活动（基于 Mapbox + 你的实时位置）
- 6 种活动类型：学习 / 运动 / 桌游 / 美食 / 旅行 / 其他
- 6 种状态机：PENDING_REVIEW → RECRUITING → FULL → STARTED → ENDED / REJECTED / CANCELED
- 活动后互评 1-5 星
- 微信原生推送 + iOS APNs + Android FCM + 国内 TPNS

## 2. 怎么注册？

3 种方式，按你方便选：

- **微信登录**（小程序、含微信 SDK 的 App 客户端）—— 1 键登录，无需填表
- **手机号**（iOS / Android）—— 收 1 条 SMS 验证码（120s 有效）
- **Apple ID**（iOS）—— 隐私优先，可隐藏真实邮箱

注册后系统会**强制**让你选至少 1 个学校 + 1 个专业 + 1 个入学年份。
这 3 项是「活动匹配」和「同校筛选」的基础，不填不让进首页。

**手机号 / 微信 unionid** 任何 1 个已注册就视为同一账号——你换
登录方式不会丢数据。

## 3. 怎么找附近的活动？

打开 App，地图模式（`📍 附近`）会默认按你当前 GPS 定位显示 50 km
半径内所有 `RECRUITING` / `FULL` 状态的活动。

**精确控制半径**：地图右下角拖动「半径滑杆」5 km / 10 km / 25 km /
50 km。

**精确控制时间**：顶部 Tab 切换 `今天` / `本周` / `本周末` / `自定义`
（自定义支持选日期范围）。

**筛选**：左上角 🎛️ 入口，可以按活动类型、学校、是否仅看女生专场、
是否免费筛选。

> **注意**：首次打开会请求「位置」权限。如果拒绝，地图会落到一个默认
> 位置（你注册时填的学校经纬度），可以手动搜索地址。

## 4. 怎么发起一个活动？

进入首页 → 底部 Tab `+` 按钮 → 选活动类型 → 填 4 块信息：

1. **基础信息**（必填）：标题、简介（500 字内）、封面图（jpg/png，
   ≤ 5 MB，最多 3 张）
2. **时间地点**（必填）：开始时间、结束时间、地点（地图选点 + 文字
   地址）、活动半径（5/10/25/50 km）
3. **参与门槛**（必填）：人数上限（2-50）、是否需要审核、是否仅
   女生、是否免费
4. **补充**（选填）：标签（最多 5 个）、备注

提交后活动进入 `PENDING_REVIEW` 状态（机器人内容安全审核 + 人工
二次审核，15 分钟内出结果），通过后变 `RECRUITING` 并出现在地图上。

> **审核不通过怎么办？** 你会收到一条系统消息 + 审核原因。
> 改完可以重新提交，不需要重新填表。

## 5. 内容安全审核是什么？会不会泄露我的隐私？

Pairhub 用 2 道关卡保护你和平台：

1. **自动审核**（微信 / 阿里云内容安全 API）：标题 + 简介 + 活动
   封面 3 块都过一遍文字 + 图片。命中违规关键词 / 血腥 / 涉政
   直接拒，秒级返回。
2. **人工二次审核**（平台运营）：自动通过的活动有 1% 概率被抽样
   复审。复审通常 < 2 小时。

**你的隐私**：

- 自动审核只把「标题 + 简介 + 封面」3 块发到 API，**不会**发
  你的个人信息（手机号、openid、IP、设备 ID 都不发）
- 审核结果只你和平台运营可见，其他用户看不到
- 同一活动重复提交 3 次不过，自动转人工（不是机器判的）

详细技术说明见 `docs/ops/mp-audit-config.md`。

## 6. 怎么报名 / 取消 / 签到？

**报名**：活动详情页 → `报名` 按钮 → 填写 1 行「对活动发起人说的话」
（100 字内，可空）→ 提交。如果活动需要审核（`isAudited = true`），
你的报名会先变 `PENDING`，等发起人 / 平台过审才变 `APPROVED`。

**取消**：我的 → 我报名的 → 找到活动 → `取消报名`。**活动开始
前 2 小时**才允许取消（避免临时放鸽子）；超过 2 小时要联系发起人
手动处理。

**签到**：活动现场发起人会展示一个 6 位签到码（动态生成，5 分钟
刷新一次）。参与者输入签到码即视为到场，状态变 `ATTENDED`。
**未签到**的报名会在活动结束后自动 CANCELED（不计入信用分）。

## 7. 活动后怎么互评？评什么？

活动状态变 `ENDED` 后 7 天内，你和所有签到参与者可以互评：

- 评分：1-5 星（必填）
- 标签：可选「准时 / 健谈 / 靠谱 / 有趣 / 细心」≤ 3 个
- 文字评论：500 字内，可空

**评完的可见性**：

- 你的「收到的总评分」和「收到的标签云」会显示在你的公开 profile
  上（其他用户可见）
- 文字评论默认**只有你和被评者可见**；如果双方都公开，才会展示
  在 profile 上
- 同一活动同一人你只能评 1 次（改评要走客服）

> **注意**：你不能给自己评；不能评没签到的人；不能评 `ENDED` 状态
> 之前/之后 7 天窗口外的人。

## 8. 怎么举报一个用户 / 一条评价 / 一个活动？

任何用户、评价、活动详情页都有「🚩 举报」按钮。

**举报类型**（5 选 1）：

- 骚扰 / 辱骂
- 欺诈 / 收款不履约
- 虚假信息（活动 / 评价）
- 涉政 / 色情 / 暴力
- 其他（必填 200 字内说明）

举报后 24 小时内会有平台运营 1 次响应。**紧急情况**（线下人身
安全威胁）请直接拨打当地 911 / 999 等紧急电话，平台无法替你
做应急响应。

平台对举报人信息**严格保密**——被举报者看不到是谁举报的。

## 9. 价格、保险、数据安全

**价格**：Pairhub v1.0 完全**免费**。平台不收服务费、不抽成。
活动本身收费与否由发起人决定（活动详情页会明示），我们只提供
「AA 收款 / 微信收款」等工具，不经手资金。

**保险**：Pairhub 平台**不**为活动参与者提供商业保险。强烈
建议所有参与者**自行购买**旅行 / 户外 / 短期意外险。如果你
策划的活动涉及高风险项目（滑雪、攀岩、户外探险等），请在活动
描述里明示「参与者需自备保险」。

**数据安全**：

- 你的账号数据存在 AWS 法兰克福（eu-central-1）+ us-east-1 灾备
- 全部 HTTPS / TLS 1.3；敏感字段（手机号、openid）AES-256 静态加密
- 备份保留 30 天滚动；30 天前的备份自动销毁
- 我们**不卖**你的数据给第三方；**不接**广告 SDK；**不嵌入**
  任何统计 / 追踪脚本

详细技术说明见 `docs/ops/legal/privacy-policy.zh.md`。

## 10. 怎么注销账号 / 删数据？

**App 内**：`我的` → `设置` → `账号与安全` → `注销账号` → 二次确认
（输 6 位手机验证码）→ 立即冻结 7 天（冷静期）→ 7 天后**完全删除**
（不可恢复）。

**删除范围**：

- 账号本身
- 头像 / 昵称 / 个人简介
- 你发的活动、报的名、写的评价
- 你的设备 token（推送）
- 你的位置历史（系统从未持久化你的位置历史，**本来就没有**）

**保留范围**（法律要求）：

- 财务相关记录（如未来涉及收费）保留 7 年
- 涉嫌违法的活动 / 评价记录，按平台合规要求保留至案件终结
- 你的活动签到记录（仅活动内部）保留 90 天后脱敏（仅留活动 ID
  和你报名的状态）

注销前可以一键导出所有个人数据（JSON 格式，≤ 7 天发到你注册邮箱）。
