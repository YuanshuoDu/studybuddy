# Pairhub 隐私政策

> 简体中文版 ｜ 英文版见 [privacy-policy.en.md](./privacy-policy.en.md)
> 最近更新：2026-06-08
> 适用产品：Pairhub 微信小程序 + iOS / Android App

我们（XXX 科技有限公司）非常重视您的隐私。本政策说明我们如何收集、使用、存储、共享和保护您的个人信息。

## 1. 我们收集哪些信息

### 1.1 您主动提供的信息

| 信息类型 | 场景 | 是否必填 |
| --- | --- | --- |
| 微信 OpenID / Apple Sub / Google Sub | 首次登录 | 必填 |
| 昵称 | 个人资料 | 必填 |
| 头像 | 个人资料 | 可选 |
| 学校 / 专业 / 年级 | 个人资料 | 可选 |
| 手机号 | 同校匹配 | 可选 |
| 地理位置 | 创建活动时获取附近位置 | 可选 |
| 活动内容（标题/描述/标签） | 您发布的活动 | 必填 |
| 评价（评分/评论） | 活动结束后 | 可选 |

### 1.2 我们自动收集的信息

- 设备信息：设备型号、操作系统版本、设备 ID
- 网络信息：IP 地址、网络类型
- 日志信息：访问时间、停留时长、错误日志

## 2. 我们如何使用信息

我们使用收集到的信息用于：

1. **账号体系**：注册、登录、身份验证
2. **核心功能**：活动发布、报名、匹配、评价
3. **内容安全**：调用微信 `msg_sec_check` API 审核您发布的内容（issue #26）
4. **改进服务**：通过脱敏数据分析用户行为，优化产品体验
5. **客户支持**：响应您的咨询和投诉
6. **法律义务**：配合司法/行政机关的合法要求

## 3. 信息的存储与保护

### 3.1 存储位置

- **中国大陆境内**：阿里云 / 腾讯云（深圳 + 上海双中心，异地容灾）
- **境外**（仅在您主动选择"海外模式"时）：AWS 法兰克福 (eu-central-1) + 弗吉尼亚 (us-east-1)

### 3.2 存储期限

| 信息类型 | 存储期限 |
| --- | --- |
| 账号基本信息 | 账号注销后 **30 天**全量清除 |
| 活动 / 评价内容 | 账号注销时同步清除 |
| 操作日志 | **180 天**（法定要求） |
| Access Token | **15 分钟**（JWT 过期） |
| Refresh Token | **30 天**（滑窗续期） |

### 3.3 安全措施

- 传输层：全站 HTTPS / TLS 1.3
- 存储层：密码 bcrypt 加密（cost = 12），敏感字段 AES-256-GCM 加密
- 访问控制：基于角色的最小权限（RBAC），审计日志保留 365 天
- 漏洞响应：发现安全漏洞请发送至 security@pairhub.app，48h 内确认

## 4. 信息共享与披露

我们**不会**出售您的个人信息。仅在以下情况共享：

1. **必要共享**：与您报名的活动组织者共享您的昵称和头像（仅在该活动范围内）
2. **同校匹配**：在您主动开启"同校匹配"功能时，与同校其他用户共享学校 / 专业信息
3. **服务提供方**：仅向提供云服务的供应商（阿里云 / 微信）共享必要的存储 / 鉴权信息
4. **法律要求**：法院、公安机关等有权机关的合法要求
5. **用户同意**：经您明确同意的其他情况

## 5. 您的权利

您对自己的个人信息享有以下权利：

| 权利 | 行使方式 |
| --- | --- |
| **查询 / 复制** | 个人中心 → 设置 → 数据导出（json / csv 格式） |
| **更正** | 个人中心 → 编辑资料 |
| **删除** | 个人中心 → 设置 → 注销账户（30 天内全量清除） |
| **撤回授权** | 设置 → 隐私 → 关闭对应功能（同校匹配 / 位置 / 手机号） |
| **投诉举报** | 举报按钮（在每条活动 / 个人主页），或邮箱 report@pairhub.app |
| **个人信息保护投诉** | 深圳市互联网信息办公室 / 国家网信办 12377 |

## 6. Cookie 与本地存储

### 6.1 微信小程序

微信小程序运行在微信客户端内，**不使用**传统 Cookie。我们使用 `wx.setStorageSync` 做本地存储，仅存储：
- 登录态（accessToken + refreshToken）
- 用户偏好（如"是否开启同校匹配"）

### 6.2 iOS / Android App

- iOS：使用 `Keychain` 存储 token，使用 `UserDefaults` 存储偏好
- Android：使用 EncryptedSharedPreferences 存储 token，使用 SharedPreferences 存储偏好

## 7. 第三方 SDK

我们使用以下第三方 SDK，每个 SDK 仅收集其功能所必需的信息：

| SDK | 提供方 | 用途 | 收集信息 |
| --- | --- | --- | --- |
| 微信开放平台 | 腾讯 | 登录 / 分享 / 支付 | OpenID、UnionID、昵称、头像 |
| Apple Sign In | Apple | iOS 登录 | Apple Sub、昵称、邮箱（可选） |
| Google Sign In | Google | Android 登录 | Google Sub、昵称、邮箱（可选） |
| Mapbox | Mapbox Inc. | 地图 / 定位 | 地理位置（仅创建活动时） |
| 阿里云 OSS | 阿里云 | 图片 / 文件存储 | 上传的文件 |
| 微信 `msg_sec_check` | 腾讯 | 内容审核 | 文本内容 |

## 8. 未成年人保护

Pairhub 面向 **18 周岁及以上**的成年用户。注册时强制勾选「我已年满 18 周岁」。

我们**不会**主动收集未成年人信息。如发现未满 18 周岁的用户，请联系 support@pairhub.app，我们将立即删除其账户。
